# Portrait image

Place your headshot here, for example:

```text
static/assets/img/archisman.jpg
```

Then set this in `content/site.toml`:

```toml
photo_path = "/assets/img/archisman.jpg"
photo_alt = "Portrait of Archisman Dutta"
```

Use a square crop if possible. A 900×900 or 1200×1200 JPEG/WebP is more than enough.
